# Laravel MySQL Project

A Laravel application built with MySQL database.

## Tech Stack
- Laravel 12
- PHP
- MySQL
- Vite
- Blade

## Installation

1. Clone repository
git clone https://github.com/Nirojiha26/laravel-mysql-project.git

2. Install dependencies
composer install

3. Copy environment file
cp .env.example .env

4. Generate application key
php artisan key:generate

5. Run migrations
php artisan migrate

6. Start server
php artisan serve
